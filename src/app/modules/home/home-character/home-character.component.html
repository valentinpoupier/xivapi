Search Character

Search for a character by name and server, and click on "details" of character to view more details.

Zodiark is server by default but you can change it to any other server or choose "any" to search on all servers.

There is currently no way to change the amount of results back returned. It will always be 50 per page with a maximum of 20 pages. This is due to how Lodestone works.

Details Character

Display the details of a character. This includes the job, gear, stats, minions, mounts and
free company (click on the name to view the free company)
